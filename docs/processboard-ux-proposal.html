Summary of all changes

# Area Change Impact
1 Multi-lane clarity "Primary lane" badge at card top Users understand why WO appears in 2+ lanes
2 Ownership Avatar + name, red for unassigned Clear who is responsible per lane
3 Empty lanes Show all 7 lanes, grayed when empty Full process visible, no hidden stages
4 Next action Boxed CTA with "Nästa steg" label Action is the most prominent element
5 Overdue / blockers Red border + blocker banner + bold badges Impossible to miss urgent issues
6 Material status Always show section, mini bar when >0 Ready for migration, shows 0 state clearly
No schema changes. No data changes. All changes are in ProcessBoard.jsx only.
